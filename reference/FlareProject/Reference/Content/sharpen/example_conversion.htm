Example Conversion


Now that we have a script to install sharpen we can try to do a simple conversion.

For example let's create a new project SharpenTutorial with PersistentExample.Java and Pilot.Java classes from reference query examples. Try to run PersistentExample in Eclipse to make sure that the project builds correctly.

As the next step we should add sharpen.properties and sharpen-common.xml files from the previous chapter to SharpenTutorial.

Now we are ready to create a build script, which will convert our java project to c#:

Build-Net.Xml
01<?xml version="1.0"?>
02<project name="Db4objects.Db4odoc" default="sharpen-docs">
03  <property file="sharpen.properties" />
04  <import file="sharpen-common.xml" />
05
06  <taskdef name="updatecsharpproject" classname="com.db4o.devtools.ant.UpdateCSharpProjectAntTask">
07    <classpath>
08      <pathelement location="lib" />
09      <path path="${path.classpath.full}" />
10    </classpath>
11  </taskdef>
12
13  <!-- Main target -->
14  <target name="sharpen-docs" depends="install-sharpen-plugin, clean">
15
16    <property name="target.dir" location="sharpen" />
17
18    <!-- Copy java files to the resource folder -->
19    <copy todir="${target.dir}/sharpened_examples/src">
20      <fileset dir="src">
21        <include name="**/*.java" />
22      </fileset>
23    </copy>
24
25        <!-- Sharpen java files -->
26    <sharpen workspace="${target.dir}" resource="sharpened_examples/src">
27      <args>
28        <!-- classpath needed for java sources compilation -->
29        <arg value="-cp" />
30        <arg path="lib/db4o-7.2.37.10417-java5.jar" />
31        <!-- Sharpen options are defined in a separate file -->
32        <arg value="@sharpen-all-options" />
33      </args>
34    </sharpen>
35
36    <!-- Define locations for the converted resources -->
37    <property name="dir.proj" location="../sharpen-tutorial.net/Db4objects.Db4odoc" />
38    <property name="dir.proj.src" location="../sharpen-tutorial.net/Db4objects.Db4odoc" />
39    <property name="dir.sharpen.src" location="sharpen/sharpened_examples.net" />
40
41        <!-- Task for copying converted resources -->
42    <macrodef name="copy-sharpened-sources">
43      <attribute name="dir" />
44      <element name="files" />
45      <sequential>
46        <copy todir="@{dir}">
47          <files />
48        </copy>
49      </sequential>
50    </macrodef>
51
52    <!-- Copy sharpened file to the destination folder -->
53    <copy-sharpened-sources dir="${dir.proj.src}">
54      <files>
55        <fileset dir="${dir.sharpen.src}/src">
56          <include name="**/*.cs" />
57        </fileset>
58      </files>
59    </copy-sharpened-sources>
60
61    <!-- Define c# resources -->
62    <fileset id="core.net.files" dir="${dir.proj}">
63      <include name="**/*.cs" />
64    </fileset>
65
66    <!-- If we already have a csproj file, we can just update it to include 
67    all the added c# resources -->
68    <updatecsharpproject projectfile="${dir.proj}/sharpen-examples.csproj">
69      <sources refid="core.net.files" />
70    </updatecsharpproject> 
71
72  </target>
73
74  <!-- Remove resources from the previous convertion run -->
75  <target name="clean" description="Delete all generated files">
76    <delete failonerror="false" includeemptydirs="true" description="Removing all generated files">
77      <fileset dir="../reference.cs.net">
78        <include name="**/*" />
79        <exclude name="**/*.csproj" />
80        <exclude name="**/*.sln" />
81      </fileset>
82    </delete>
83    <delete failonerror="false" includeemptydirs="true" description="Removing all generated files">
84      <fileset dir="sharpen">
85        <include name="**/*" />
86        <exclude name="**/lib/*" />
87      </fileset>
88    </delete>
89  </target>
90</project>

As you can see from the build script the sequence of actions is the following:

  1. Install sharpen plugin
  2. Clean up the destination folder
  3. Sharpen java resources and copy the resulted c# files to the destination

sharpen-all-options file provides options for the conversion:

@sharpen-collections-mapping

-pascalCase+
-nativeTypeSystem
-nativeInterfaces


-fullyQualify File

-methodMapping System.out.println System.Console.WriteLine


-typeMapping com.db4o.Db4o Db4objects.Db4o.Db4oFactory
-typeMapping com.db4o.ext.ExtDb4o Db4objects.Db4o.Ext.ExtDb4oFactory

-namespaceMapping com.db4o Db4objects.Db4o
-namespaceMapping com.db4odoc Db4objects.Db4odoc

-namespaceMapping java Sharpen

-propertyMapping java.lang.System.out Sharpen.Runtime.Out
-propertyMapping java.lang.System.err Sharpen.Runtime.Err
 

It also references another options file for an easier structuring:


-methodMapping java.util.Iterator.hasNext HasNext
-methodMapping java.util.Iterator.next Next

-typeMapping com.db4o.ObjectSet Db4objects.Db4o.IObjectSet
-methodMapping com.db4o.ObjectSet.hasNext Db4objects.Db4o.IObjectSet.HasNext
-methodMapping com.db4o.ObjectSet.next Db4objects.Db4o.IObjectSet.Next

-typeMapping java.util.Comparator System.Collections.IEnumerator


-typeMapping java.util.ArrayList System.Collections.ArrayList
-methodMapping java.util.List.addAll AddRange
-methodMapping java.util.AbstractList.addAll AddRange
-methodMapping java.util.ArrayList.addAll AddRange
