Add Custom Configuration

When your database uses special configuration items, like another string encoding, special typehandler etc you need to configure that in OME. OME is configured by loading a specially build configuration assembly.

  1. 1. Create a class library project which contains all your custom configurations.
  2. Add a reference to the OMCustomConfigurationforUser assembly. This library you can get from the path where OME is installed. Generally it is in C:\Program Files (x86)\db4o\db4o-X.X\Object Manager Enterprise
  3. To add configuration to an embedded database inherit from IOMNEmbeddedCustomConfigurationProvider. Otherwise, when your connecting to a server inherit from IOMNClientCustomConfigurationProvider:
  4. Compile this project.Then load it into OMN using "Custom Config Assembly Path"
  5. Load all the other associated assemblies which are be needed by your custom config. Or assemblies with the classes and types stored in the database.
