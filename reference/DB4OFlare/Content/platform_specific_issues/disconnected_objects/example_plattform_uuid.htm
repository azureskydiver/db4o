Example

This example demonstrates how you can use s to identify objects across objects containers. Take a look advantages and disadvantages of s: See "Comparison Of Different IDs"

Don't forget to add UUID support.

This example assumes that all object have a common super-class, IDHolder, which holds the in a field.

It's important to index the id-field, otherwise looking up for object by id will be slow.

The id is hold by the object itself, so you can get it directly.

You can get the object you can by a regular query.
