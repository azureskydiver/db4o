Common Configuration Items

Configuration items add special capability to the system. Here's a list of all configuration items available.

TransparentActivationSupport

Support for transparent activation. See "Transparent Activation Framework"

TransparentPersistenceSupport

Support for transparent persistence. See "Transparent Persistence"

UniqueFieldValueConstraint

Set up unique Field constraints. See "Unique Constraints"

JavaSupport

Adds support for reading a Java db4o-database. Add this item when you need to read a database which contains stored Java-objects. See "Cross-Platform Applications"

DotnetSupport

Adds support for reading a .Net db4o-database. Add this item when you need to read a database which contains stored .Net object. See "Cross-Platform Applications"

DotNetSupportClientServer

Adds support for a talking with a .NET client / server . See "Cross-Platform Applications"

BigMathSupport

Add support for the BigDecimal and BigInteger classes. See "BigMath"

FreespaceMonitoringSupport

Enables you to monitor the free-space-manager. See "Runtime Statistics"

IOMonitoringSupport

Enables you to monitor the IO-activity of db4o. See "Runtime Statistics"

NativeQueryMonitoringSupport

Enables you to monitor the native queries. See "Runtime Statistics"

NetworkingMonitoringSupport

Enables you to monitor the network activity of db4o. See "Runtime Statistics"

ObjectLifecycleMonitoringSupport

Enables you to monitor the object lifecycles. See "Runtime Statistics"

QueryMonitoringSupport

Enables you to monitor db4o queries. See "Runtime Statistics"

ReferenceSystemMonitoringSupport

Enables you to monitor db4o's reference system. See "Runtime Statistics"
