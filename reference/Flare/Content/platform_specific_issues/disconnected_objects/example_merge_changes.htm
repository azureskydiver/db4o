Example Merge Changes

This example shows how changes are merged from the disconnected object to the object to update. Basically it traverses the object-graph. The value types are just copied over. All reference-types are first checked if they're existing object. If it is,the primitives are copied over, otherwise it's a new object.

Utilizing reflection this approach can be automated. You can also use existing libraries like Automapper Dozer which help you to do this.
