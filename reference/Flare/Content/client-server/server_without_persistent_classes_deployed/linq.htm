LINQ

Db4o LINQ is an optimisable query: under the hood it is converted to SODA where possible. However, SODA equivalents may not exist for some more complex LINQ (like aggregates). In this case the query will run against all object instances to return the correct result. In case when class definitions are not available - this is not possible.
