Monitor Network

You can monitor the network-operations of db4o in client-server mode.

Configure the Network Monitoring Support

First you need to add the monitoring support to the db4o configuration. The network monitoring support is in two separate configuration-items. The NetworkingMonitoringSupport can be used on the server and on a client to monitor the network statistics of the server or client.

The ClientConnectionsMonitoringSupport can be added to the db4o server to monitor the connected clients.

The Network Statistics

BytesReceivedPerSecondnetwork bytes sent/sec: Tells you how many bytes this client or server has receives per second.

BytesSentPerSecondnetwork bytes received/sec: Tells you how many bytes this client or server sends per second.

MessagesSentPerSecondnetwork messages sent/sec: Tells you how many messages this client or server sends per second.

ConnectedClientCountnumber of connected clients: Tells you how many clients are connected to this server.
