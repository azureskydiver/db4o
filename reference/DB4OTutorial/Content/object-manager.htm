Object Manager

The object manager allows us to open and explore a database.

Installation.

The object manager is an Eclipse plugin. In the db4o distribution there's a subfolder 'ome', which contains a zip-file. We unzip that file and then start up Eclipse. There we choose 'Help'->'Install New Software...'. Then we add the location of the unzipped object manage as an update site and continue the installation.

The object manager is a Visual Studio plugin. In the db4o distribution there's a subfolder 'omn', which contains the setup for it. We install it and then open Visual Studio.

Using the Object Manager

To use the object manager we first open the OME perspective ('Window'->'Open Perspective'->'Other...'. Then we can open a db4o database.

After the installation we can open db4o databases with Visual Studio (Tools->'ObjectManager Enterprise'->Connect). After that we can explore the database.

After connecting to the database we see a list of stored classes on the right. Below that we can inspect the a stored class and see it details.

To query the database we drag and drop the fields we're interested in to the query builder. Then we can change the condition and values on that query and run it.

On the result page we can select the different objects. And inspect their values. We also can change values of objects and then store the data.
