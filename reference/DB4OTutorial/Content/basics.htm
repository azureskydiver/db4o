Basics

Let's start with the basics. For this first introduction we create a tiny domain model, consisting out of drivers and cars.

Open a Database

For all operations we need to open the database first. To open the database we pass a file-path to db4o. db4o will open or create that database an return an object container. The object container is the connection to our database. We do all our operations through the object container.

Store an Object

To store an object in the database, we simply call the store-method on the object container.

Query the Database

After storing objects we want to query for our stored objects. We can do this with .

Update an Object

In order to update an object we first query for it. Then we change it and store it again. db4o will recognize the updated object and store the changes in the database.

Delete an Object

Deleting an object is easy. First we query for the object and then delete it.
