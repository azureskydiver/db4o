Basics

Lets start with the basics. For this tutorial we create a tiny domain model, consisting out of drivers and cars. (Click here to see the domain model classes)

Open a Database

For all operations we first need to open the database. To open the database we pass a file-path to db4o. db4o will open or create that database an return a object container. The object container is the connection to our database. We do all our operations through the object container.

Store an Object

To store a object in the database, we simply call the store-method on the object container.

Update an Object

In order to update an object we first query for it. Then we change it and store it again. db4o will recognize the updated object and store the changes in the database.

Delete an Object

Deleting an object is easy. First we query for the object and then delete it.
