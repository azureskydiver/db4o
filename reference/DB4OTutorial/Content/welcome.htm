Getting db4o

There are two way how you can get db4o. Either you can download a distribution from db4o website, or you can use MavenNuGet to get it.

Downloading db4o

Download db4o from the developer website. I recommend to use the 'production' version of db4o. Choose the download which fits your environment. The download contains the binaries, documentation and source.

Using Maven

Another way to grab db4o is via Maven. For the tutorial we just grab the full db4o distribution.

<dependency>
    <groupId>com.db4o</groupId>
    <artifactId>db4o-full-java5</artifactId>
    <version>8.1-SNAPSHOT</version>
</dependency>

Additionally we need to add the db4o Maven repository.

<repositories>
   <repository>
        <id>db4o</id>
        <name>Db4o</name>
        <url>https://source.db4o.com/maven/</url>
   </repository>
</repositories>

Using NuGet

Another way to grab db4o is with NuGet. I recommend you to use the 'db4o-devel'-package for this tutorial. Open the NuGet console and use the install-command:

Install-Package db4o-devel
