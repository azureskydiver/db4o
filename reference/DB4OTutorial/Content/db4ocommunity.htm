Report Bugs

Software contains bugs and so does db4o. If you find bugs please report them to us. In case you're not sure if you really found a bug, you maybe want to discuss your issue first on the db4o forum. As soon as you're sure that you found a bug it best to report in our bug-tracking system. The login is the same as in the db4o forums.

When possible include a small test program which reproduces the issue with the bug report. A test case avoids confusion and effort to reproduce the bug.
