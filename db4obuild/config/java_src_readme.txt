This folder contains the source code of db4o and associated projects.

bloat               - byte code optimization library 
                      (http://developer.db4o.com/Resources/edit.aspx/Working_With_Sources/www.cs.purdue.edu/s3/projects/bloat/)
db4oj               - db4o core (JDK1.1)
db4oj.tests         - collection of db4o regression tests.
db4ojdk1.2          - db4o sources for JDK1.2-1.4 
db4ojdk5            - db4o sources for JDK 5-6
db4onqopt           - db4o native query optimization library
db4ounit            - unit test framework for db4o 
                      (http://developer.db4o.com/Resources/view.aspx/Reference/Db4o_Testing_Framework) 
db4ounit.extensions - Db4oUnit extensions for testing db4o 
                      (http://developer.db4o.com/Resources/view.aspx/Reference/Db4o_Testing_Framework)  

For more information see: 
http://developer.db4o.com/Resources/view.aspx/Working_With_Source_Code
