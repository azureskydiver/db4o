ObjectField Configuration

ObjectField/IObjectField is an interface providing configuration methods for class fields. ObjectField instance can be obtained through:

.NET:

IObjectField of = Db4oFactory.Configure().ObjectClass(clazz).ObjectField("fieldName");


.NET:

public void CascadeOnActivate(bool flag);

sets cascaded activation behavior. For more information see Activationcreate it.


.NET:

public void CascadeOnDelete(bool flag);

sets cascaded delete behavior. For more information see Delete_Behaviorcreate it.


.NET:

public void CascadeOnUpdate(bool flag);

sets cascaded update behavior. For more information see Update_Depthcreate it.


.NET:

public void Indexed(bool flag);

turns indexing on or off. For more information see Indexingcreate it.


.NET:

public void Rename(string newName);

renames a field of a stored class. For more information see Refactoring_And_Schema_Evolutioncreate it.


.NET:

public void QueryEvaluation (bool flag);

toggles query evaluation.

All fields are evaluated by default. Use this method to turn query evaluation off for specific fields.
