Sorting Query Results

Often applications need to present query results in a sorted order. There are several ways to achieve this with db4o.

This Pilot class will be used in the following examples:

SortingExample.vb: SetObjects
01Public Shared Sub SetObjects()Sub SetObjects()
02            File.Delete(YapFileName)
03            Dim db As IObjectContainer = Db4oFactory.OpenFile(YapFileName)
04            Try
05                Dim i As Integer = 0
06                While i < 10
07                    Dim j As Integer = 0
08                    While j < 5
09                        Dim pilot As Pilot = New Pilot("Pilot #" + i.ToString(), j + 1)
10                        db.Set(pilot)
11                        System.Math.Min(System.Threading.Interlocked.Increment(j), j - 1)
12                    End While
13                    System.Math.Min(System.Threading.Interlocked.Increment(i), i - 1)
14                End While
15            Finally
16                db.Close()
17            End Try
18        End Sub
