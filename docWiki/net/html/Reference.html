Reference

db4o (database for objects) is a high-performance, embeddable open source object database. Db4o supports Java and .NET platforms, thus giving you a wide selection of implementation choices.

This reference is for anyone who is using or is going to use db4o and wants to have handy and thorough information on most common db4o programming questions.

If you are a beginner with db4o we recommend you to start with the Tutorial, which can be found in the distribution doc folder. This reference uses an approach, which assumes that the reader has a basic understanding of db4o programming. Each chapter starts with a brief introduction followed by detailed reference information including lots of meaningful examples.
