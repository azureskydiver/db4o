Object Manager Enterprise

Starting from db4o version 7.8 db4o distribution package includes Object Manager Enterprise - graphical tool for browsing and managing db4o databases. The tool is available in 2 flavours: Eclipse plugin for Java users and Visual Studio 2005/2008 plugin for .NET users.
