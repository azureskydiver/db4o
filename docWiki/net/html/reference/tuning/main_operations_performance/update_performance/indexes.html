Indexes

Updating indexed fields always takes longer as the index should be updated as well. This is shown in the following test:

UpdatePerformanceBenchmark.cs: RunIndexTest
01private void RunIndexTest()
02        ...{
03            System.Console.WriteLine("Update test for objects with and without indexed fields");
04
05            int objectsToUpdate = 100;
06            Init();
07            System.Console.WriteLine("Updating " + objectsToUpdate + " of " + _count + " objects");
08            Clean();
09            Open(Configure());
10            Store();
11            UpdateItems(objectsToUpdate);
12            Close();
13
14            Clean();
15            Init();
16            System.Console.WriteLine("Updating " + objectsToUpdate + " of " + _count + " objects with indexed field");
17            Open(ConfigureIndexTest());
18            Store();
19            UpdateItems(objectsToUpdate);
20            Close();
21        }
