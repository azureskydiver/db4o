Update Performance

Update performance is influenced by the similar factors as Insert Performance. The main factors include: configuration, disk access times, complexity of objects.

The following chapters provide some simple tests showing the influence of the above-mentioned factors. The test results are provided for Toshiba Sattelite Pro A120 notebook with 1,5Gb RAM 120GB ATA drive running on Vista and may be different on a different environment.

The following Item class is used in most of the tests:

226        private IConfiguration Configure()
227        ...{
228            IConfiguration config = Db4oFactory.NewConfiguration();
229            // using MemoryIoAdapter improves the performance

237        private IConfiguration ConfigureTP()
238        ...{
239            IConfiguration config = Db4oFactory.NewConfiguration();

293        private void Store()
294        ...{
295            StartTimer();
296            for (int i = 0; i < _count; i++)
297            ...{
298                Item item = new Item("level" + i, null);

300                ...{

431        private void Open(IConfiguration config)
432        ...{
433            if (_isClientServer)
434            ...{
435                int port = TCP ? PORT : 0;
436                string user = "db4o";
437                string password = user;
438                objectServer = Db4oFactory.OpenServer(_filePath, port);
439                objectServer.GrantAccess(user, password);
440                objectContainer = TCP ? Db4oFactory.OpenClient(_host, port, user,
441                        password) : objectServer.OpenClient();
442            }
443            else
444            ...{
445                objectContainer = Db4oFactory.OpenFile(config, _filePath);
446            }
447        }
448        // end Open
