Server Configuration

Server configuration interface privides access to the server configuration. This interface allows access to all common configuration methods through common() method (Java) or Common property (.NET). In addition to common configuration the following methids are available:

.NET:

public ICacheConfiguration Cache {get;}

Returns cache configuration.

.NET:

IFileConfiguration File {get;};

Returns file configuration, giving access to file-related configuration methods.
