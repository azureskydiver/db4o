Batch Messaging Example

Let's create a small example to test batch messaging mode behavior. We will use bulk insert with and without batch messaging configuration.

BatchExample.cs: FillUpDb
01private static void FillUpDb()
02        ...{
03            IClientConfiguration configuration = Db4oClientServer.NewClientConfiguration();
04            // Uncomment the following line to test CS mode without batching
05            // configuration.Networking.BatchMessages = false;
06            // Uncomment the following line to test maximum batch queue size (batch mode should be enabled)
07            // configuration.Networking.MaxBatchQueueSize = 1024;
08            IObjectContainer container = Db4oClientServer.OpenClient(configuration, Host, Port, User,
09                        Password);
10            try
11            ...{
12                Console.WriteLine("Testing inserts");
13                DateTime dt1 = DateTime.UtcNow;
14                for (int i = 0; i < NoOfObjects; i++)
15                ...{
16                    Pilot pilot = new Pilot("pilot #" + i, i);
17                    container.Store(pilot);
18                }
19                DateTime dt2 = DateTime.UtcNow;
20                TimeSpan diff = dt2 - dt1;
21                Console.WriteLine("Operation time: " + diff.TotalMilliseconds + " ms.");
22            }
23            finally
24            ...{
25                container.Close();
26            }
27        }
BatchExample.vb: FillUpDb
01Private Shared Sub FillUpDb()Sub FillUpDb()
02            Dim configuration As IClientConfiguration = Db4oClientServer.NewClientConfiguration()
03            ' Uncomment the following line to test CS mode without batching
04            ' configuration.Networking.BatchMessages = False
05            ' Uncomment the following line to test maximum batch queue size (batch mode should be enabled)
06            ' configuration.Networking.MaxBatchQueueSize = 1024
07            Dim container As IObjectContainer = Db4oClientServer.OpenClient(configuration, Host, Port, User, Password)
08            Try
09                Console.WriteLine("Testing inserts")
10                Dim dt1 As DateTime = DateTime.UtcNow
11                Dim i As Integer = 0
12                While i < NoOfObjects
13                    Dim pilot As Pilot = New Pilot("pilot #" + i.ToString(), i)
14                    container.Store(pilot)
15                    System.Math.Min(System.Threading.Interlocked.Increment(i), i - 1)
16                End While
17                Dim dt2 As DateTime = DateTime.UtcNow
18                Dim diff As TimeSpan = dt2 - dt1
19                Console.WriteLine("Operation time: " + diff.TotalMilliseconds.ToString() + " ms.")
20            Finally
21                container.Close()
22            End Try
23        End Sub
You can run this example with the server from Simple db4o Server chapter. In order to see the influence of batch mode configuration try commenting and uncommenting line disabling batch mode. In addition you can try different values of NO_OF_OBJECTS constant.

If the value of NO_OF_OBJECTS is high (>1,000,000) you may notice that the memory consumption increases a lot. In order to decrease it, uncomment the line setting the maximum batch queue size.
