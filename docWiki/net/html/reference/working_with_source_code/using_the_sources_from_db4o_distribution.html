Using The Sources From Db4o Distribution

The source code is available in /src folder in your db4o distribution.

In .NET version the following projects are available:

Db4oAdmin

db4o command-line utility. For a list of functions execute Db4oAdmin.exe without any arguments.

Db4objects.Db4o

db4o core.

Db4objects.Db4o.Tests

collection of db4o regression tests.

Db4objects.Db4o.Tools

some useful db4o utilities.

Db4oTutorial

db4o tutorial sources.

Db4oUnit

unit test framework for db4o.  

Db4oUnit.Extensions

Db4oUnit extensions for testing db4o.  

Working with the sources is very straightforward: you just need to open a solution file from Microsoft Visual studio. The sources are provided for Visual Studio 2003 and 2005, for desktop applications and Compact Framework.
