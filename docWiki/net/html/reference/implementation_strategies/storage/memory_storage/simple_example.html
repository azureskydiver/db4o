Simple Example

Let's have a look at a simple example. For reuse, it is convenient to create a separate method, which will produce the necessary configuration:

MemoryStorageExample.cs: GetInMemoryConfig
1private static IEmbeddedConfiguration GetInMemoryConfig()
2        ...{
3            MemoryStorage ms = new MemoryStorage();
4            IEmbeddedConfiguration config = Db4oEmbedded.NewConfiguration();
5            config.File.Storage = ms;
6            return config;
7        }
MemoryStorageExample.vb: GetInMemoryConfig
1Private Shared Function GetInMemoryConfig()Function GetInMemoryConfig() As IEmbeddedConfiguration
2            Dim ms As New MemoryStorage()
3            Dim config As IEmbeddedConfiguration = Db4oEmbedded.NewConfiguration()
4            config.File.Storage = ms
5            Return config
6        End Function

In this method we use create a default MemoryStorage with doubling growth strategy and a new bin.

From the following code you can see that further usage is the same as with the default disk storage:

MemoryStorageExample.cs: CreateDatabase
01private static void CreateDatabase(IEmbeddedConfiguration config)
02        ...{
03            // the following line is only necessary for the database on disk
04            File.Delete(Db4oUri);
05
06            long startTime = DateTime.Now.Ticks;
07            // Db4oUri will be the name of the database file for the default
08            // configuration and the name of a MemoryBin for in-memory database
09            IObjectContainer container = Db4oEmbedded.OpenFile(config, Db4oUri);
10            try
11            ...{
12                Library lib = new Library();
13                for (int i = 0; i < ObjectCount; i++)
14                ...{
15                    lib.AddBook(new Book("title" + i, lib));
16                }
17                container.Store(lib);
18                Book book = (Book)container.QueryByExample(
19                        new Book("title1", null))[0];
20                System.Console.WriteLine(book);
21            }
22            finally
23            ...{
24                container.Close();
25            }
26            System.Console.WriteLine(String.Format("Time to create a database: {0} ticks",
27                    DateTime.Now.Ticks - startTime));
28        }
MemoryStorageExample.vb: CreateDatabase
01Private Shared Sub CreateDatabase()Sub CreateDatabase(ByVal config As IEmbeddedConfiguration)
02            ' the following line is only necessary for the database on disk
03            File.Delete(Db4oUri)
04
05            Dim startTime As Long = DateTime.Now.Ticks
06            ' Db4oUri will be the name of the database file for the default
07            ' configuration and the name of a MemoryBin for in-memory database
08            Dim container As IObjectContainer = Db4oEmbedded.OpenFile(config, Db4oUri)
09            Try
10                Dim [lib] As New Library()
11                For i As Integer = 0 To ObjectCount - 1
12                    [lib].AddBook(New Book("title" & i, [lib]))
13                Next
14                container.Store([lib])
15                Dim book As Book = DirectCast(container.QueryByExample(New Book("title1", Nothing))(0), Book)
16                System.Console.WriteLine(book)
17            Finally
18                container.Close()
19            End Try
20            System.Console.WriteLine([String].Format("Time to create a database: {0} ticks", DateTime.Now.Ticks - startTime))
21        End Sub

To demonstrate the performance improvement, please run the method above with the default configuration:
