Optimization at Query Execution Time

 
Note: This will not work with JDK1.1.

To enable code analysis and optimization of native query expressions at query query execution time, you just have to add db4o-5.x-nqopt.jar and bloat-1.x.jar to your CLASSPATH. Optimization can be turned on and off with the following configuration setting:
 
Db4o.configure().optimizeNativeQueries(boolean optimizeNQ)
