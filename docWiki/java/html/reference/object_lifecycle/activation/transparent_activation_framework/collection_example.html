Collection Example

This example will show how to use TA for classes having collections as a field. TA project provides a simple TA collection implementation in a PagedList class. This collection is a very basic implementation and most probably will be replaced with a more mature collection in the final release. However, it can give you an idea on how to work with a collection.

In order to run the following example you will need to download db4ota and db4otaj projects from db4o svn. 

We will use a Team class with a collection of Pilot objects:

32  protected void activate() ...{
33    if (_activator == null)
34      return;

39    // even simple String needs to be activated
40    activate();

42  }

Store and retrieve using the same configuration as in the previous example.

 

TAExample.java: storeCollection
01private static void storeCollection() ...{
02    new File(DB4O_FILE_NAME).delete();
03    ObjectContainer container = database(configureTA());
04    if (container != null) ...{
05      try ...{
06        Team team = new Team();
07        for (int i = 0; i < 10; i++) ...{
08          team.addPilot(new Pilot("Pilot #" + i));
