Database For OSGi

This topic applies to Java version only 

db4o_osgi project provides a service, which allows to use db4o as an OSGi plugin.

db4o_osgi first appeared in db4o version 6.3.

db4o_osgi.jar can be found in /lib folder of the Java distribution with detailed API documentation in /doc/osgi folder.
