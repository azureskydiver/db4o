Object Manager Sources

Object Manager is a GUI tool for browsing, querying and managing db4o database.  Object Manager sources are available from the following projects in our SVN (https://source.db4o.com/db4o/trunk):

  • objectmanager-swing - contains Object Manager GUI application
  • objectmanager-api - contains Object Manager API

Object Manager application depends on db4o Java library. In order to build Object Manager you will need to check out and build db4o java distribution. The easiest way to build both db4o and Object Manager is to follow instructions in Building Java Version. You will only need to uncomment the following lines in build.xml to include Object Manager:

<fileset dir="${dir.objectmanager-api}"            includes="build-1.1.xml"/>

<fileset dir="${dir.objectmanager-swing}"            includes="build-1.1.xml"/>
