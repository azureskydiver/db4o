Logging Storage

In this example we will implement a simple file base storage, which will log messages about each IO operation. In the implementation you can see that most of the functionality is derived from StorageDecorator and BinDecorator classes with additional logging added:

LoggingStorage.java
01/**//* Copyright (C) 2004 - 2009  Versant Corporation http://www.versant.com */
02package com.db4odoc.Storage;
03
04import java.util.logging.*;
05
06import com.db4o.ext.*;
07import com.db4o.io.*;
08
09public class LoggingStorage extends StorageDecorator ...{
10
11  public LoggingStorage()...{
12    // delegate to a normal file storage
13    this(new FileStorage());
14  }
15  
16  public LoggingStorage(Storage storage) ...{
17    // use submitted storage as a delegate
18    super(storage);
19  }
20
21  /** *//**
22   * opens a logging bin for the given URI.
23   */
24  @Override
25  public Bin open(BinConfiguration config) throws Db4oIOException ...{
26    final Bin storage = super.open(config);
27    if (config.readOnly()) ...{
28      return new ReadOnlyBin(new LoggingBin(storage));
29    }
30    return new LoggingBin(storage);
31  }
32
33  /** *//**
34   * LoggingBin implementation. Allows to log information
35   * for each IO operation
36   *
37   */
38  static class LoggingBin extends BinDecorator ...{
39
40    public LoggingBin(Bin bin) ...{
41      super(bin);
42    }
43
44    // delegate to the base class and log a message
45    public void close() throws Db4oIOException ...{
46      super.close();
47      Logger.getLogger(this.getClass().getName()).log(Level.INFO,
48          "File closed");
49    }
50
51    // log a message, then delegate
52    public int read(long pos, byte[] buffer, int length)
53        throws Db4oIOException ...{
54      Logger.getLogger(this.getClass().getName()).log(
55          Level.INFO,
56          String.format("Reading %d bytes and %d position", length,
57              pos));
58      return super.read(pos, buffer, length);
59
60    }
61
62    // log a message, then delegate
63    public void write(long pos, byte[] buffer, int length)
64        throws Db4oIOException ...{
65      Logger.getLogger(this.getClass().getName()).log(
66          Level.INFO,
67          String.format("Writing %d bytes and %d position", length,
68              pos));
69      super.write(pos, buffer, length);
70    }
71
72    // log a message, then delegate
73    public void sync() throws Db4oIOException ...{
74      Logger.getLogger(this.getClass().getName()).log(
75          Level.INFO,"Syncing");
76      super.sync();
77    }
78
79  }
80}

Use the LoggingStorage implementation with the following code (you can find a working example if you download LoggingStorage class).

Java:

config.file().storage(new LoggingStorage());
