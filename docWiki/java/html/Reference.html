Reference

This reference is for anyone who is using or is going to use db4o and wants to have handy and thorough information about it.

The reference is offered for Java, .NET and mono environments, providing examples in java, Visual Basic and c#. Online version goes with filters on the right pane, which can help you to concentrate on the requested content.

The information is presented in topics sets; each of them is atomic and can be used in arbitrary order according to your current needs. The same applies to the code examples which can be downloaded and tested in your environment. (Note for Java users - the examples are JDK5 compatible.)
